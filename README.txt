Folder Content
1. HomeSense - Base Application to be installed on motes connected to motion sensors.
   To make the project run
     make install.1 telosb.

2. HomeSenseBase - Base application to be installed on the mote which is connected to USB serial port of raspberry pi/pc.

3. HomeSenseJava - Eclipse project for running on Raspberry Pi or PC. Main class to start the application is -
        wustl.cse521s.PublishSubscribeSample
4. homesenseweb.rar -- web application as eclipse project that can be imported and run in Tomcat web server.

5. supporting_jars - library jar files which are required for HomeSenseJava and homesenseweb projects.
